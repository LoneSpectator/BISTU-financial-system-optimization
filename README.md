# BISTU财务系统优化脚本

优化北信科财务系统的脚本工具

> 本脚本由ChatGPT GPT-5模型编写，仅经过简单测试

针对北信科财务系统中的智能报销系统，在填写支付方式时，刷卡金额和报销金额输入框会检测禁止输入空格，而从Excel粘贴总会带一个空格，本脚本会自动删除空格。
